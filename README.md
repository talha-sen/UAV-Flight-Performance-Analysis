# UAV Flight Performance Analysis

This project presents a comprehensive flight performance analysis of a fixed-wing UAV using aerodynamic data obtained from XFLR5 and numerical calculations in Excel.

## Key Analysis
- Lift, drag, and moment characteristics
- Power required vs velocity
- Rate of climb (ROC)
- Flight envelope (V-n diagram)

## Key Results
- CLmax ≈ 1.3
- Stall angle ≈ 15
- Maximum L/D ≈ 78

## Files Included
- Flight_Performance_Analysis_Talha_SEN.pdf → Detailed analysis report
- Flight_Performance_Analysis_Tables.xlsx → Performance calculations
- Graphs → Aerodynamic curves and performance plots

## Tools Used
- XFLR5
- Microsoft Excel
